Video Controls Practice

  • Autoplay: Add autoplay to the tag to start immediately (usually requires muted).
  • Loop: Add loop to make the video restart automatically.
  • Preload: Use preload="none" (save data), metadata (only load duration/size), or auto (load everything).

Bonus: Background Video Mode

This version is often used for hero headers. It plays automatically, loops forever, and has no controls.
